📄 Termos de Uso

1. Aceitação dos Termos

Ao criar uma conta no FinanceiroApp, você concorda com estes Termos de Uso. O uso continuado do serviço implica aceitação de eventuais alterações.

2. Período de Teste

Novos usuários recebem um período de teste gratuito conforme configurado pelo administrador (padrão: 7 dias). Após o vencimento, o acesso para novos cadastros é suspenso até a assinatura.

3. Dados do Usuário

Seus dados financeiros são armazenados em uma planilha Google de sua propriedade. O FinanceiroApp não compartilha seus dados com terceiros.

4. Pagamento

O pagamento é processado via PIX. Após o pagamento, a ativação pode levar até 24h para confirmação manual pelo administrador.

5. Rescisão

O administrador pode suspender contas que violem estes termos. Dados podem ser excluídos 30 dias após o cancelamento.
